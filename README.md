Email Automation System

Email Automation System is a Python project that automates sending emails to multiple recipients efficiently and reliably. It demonstrates practical skills in SMTP, SQLite, task automation, and error handling.

---

Technologies Used

Python 3.x – main programming language

SMTP – sending emails

SQLite – storing recipients and email status

APScheduler – scheduling automated emails

Logging – tracking success/failure of emails

---

Features

Manual email sending

Automated scheduled sending (weekly/monthly)

Tracks email status (Pending, Sent, Failed)

Error handling ensures reliable delivery

Logs all sent and failed emails

Scalable to multiple recipients without code changes
